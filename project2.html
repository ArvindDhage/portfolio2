Ecommerce-Sales-Dashboard

The Flipmart E-commerce Sales Dashboard is an interactive Power BI dashboard that provides a comprehensive overview of sales performance from 2019 to 2022. It displays key metrics such as total sales (₹12.64M), total profit (₹1.47M), total quantity sold (178K), and total transactions (51.29K). Users can filter data by date and shipping mode to analyze trends in customer segments, including Consumer, Corporate, and Home Office. Visual elements like bar charts, line graphs, a world map, and pie charts highlight sales by shipping mode, monthly trends, country-wise distribution, top-performing sub-categories, and profit by category, offering valuable insights for strategic business decisions.

SignLanguage-DL-Project

The SignLanguage-DL-Project is a deep learning-based system developed to recognize and translate sign language gestures into text. Using Convolutional Neural Networks (CNNs), the model is trained on a dataset of hand signs to accurately classify alphabetic and symbolic gestures in real time. The system is capable of processing live video or image input, making it accessible for communication between hearing-impaired individuals and non-sign language users. Deployed using platforms like Streamlit or Flask, the project serves as an assistive tool to enhance inclusivity and accessibility through AI.

US-Visa-Approval-Prediction-UI

The US-Visa-Approval-Prediction-UI is a machine learning-based web application designed to predict the likelihood of US visa approval based on applicant data. Built with a user-friendly interface, the system collects key inputs such as education level, job position, salary, and employment history, and uses a trained ML model to provide real-time predictions. The application helps users, consultants, and HR departments assess visa approval chances with greater confidence. It is deployed with tools like Streamlit or Flask, and integrates a backend model trained on historical visa data to offer accurate, transparent, and fast evaluations.

/ Unit-Converter-App-Java-Android-Development

The Unit Converter App is a lightweight and user-friendly Android application developed in Java that allows users to quickly convert values across multiple units. The app supports conversions for categories such as length, weight, temperature, volume, and speed, offering real-time and accurate results. Designed with a clean UI and responsive layout, the app is ideal for students, engineers, and daily users needing quick unit conversions. It ensures a seamless experience with instant calculations, offline functionality, and intuitive navigation tailored for all age groups.
